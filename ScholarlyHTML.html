1. Abstract

Class Manager (ClaMa) este un utilitar bazat pe tehnologiile web. Gândit ca o soluție virtuală a unor procese ale învățământului precum evalurea, notarea sau partajare de resurse educaționale, ClaMa este un ajutor bine venit pentru cei ce caută suport virtual în domeniul educațional.
